# About Hive (Short)
Hive (hive.io) is a decentralized information-sharing blockchain built on the Delegated Proof of Stake (DPoS) consensus mechanism. Hive supports many different types of Web3 applications that contribute to the freedom and accessibility of data. The performance of the blockchain is designed to scale with widespread adoption of the currency and its layered solutions. By combining lightning-fast processing times and fee-less transactions, Hive provides leading user-friendly blockchain technology in a sustainable and accessible platform. Learn more about Hive and get started today at https://hive.io.
